**Document Version:** 1.0

**Status:** Active

## Purpose

This document defines the Platform repository product scope within the Fitzpatrick Family Portfolio.

---

## Portfolio Position

| Product | Repository | Scope |
|---------|------------|-------|
| Fitzpatrick Family Financial Assistant | `FamilyFinanceAssistant` | Finance |
| Fitzpatrick Family Platform | `FitzpatrickFamilyPlatform` | Infrastructure, Home Automation, Energy Management, AI Services, Shared Services, Family Intelligence |

---

## Platform Ownership

The Platform repository owns non-finance household platform capabilities.

The Platform does not own finance data, banking workflows, budgeting, transaction processing, or investments.

---

## Engineering Automation

This repository contains an independent Platform Engineering Automation Platform foundation.

It follows established portfolio patterns without sharing implementation code with the Fitzpatrick Family Financial Assistant repository.
