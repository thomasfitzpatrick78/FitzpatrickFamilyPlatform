# Documentation Standards

**Document Version:** 1.0

**Status:** Active

**Milestone:** Milestone 11

---

## Purpose

This document defines documentation standards for the Platform repository.

---

## Standards

- Authoritative documents are Markdown.
- Documents include title, version, status, milestone, purpose, related documents, and revision history when applicable.
- Links should be repository-relative.
- Historical decisions are preserved rather than rewritten.
- Finance terminology shall not be used as Platform scope.
